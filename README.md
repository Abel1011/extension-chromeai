## ğŸš€ Getting Started

### Prerequisites

- Google Chrome version 127 or higher
- Chrome's built-in AI enabled (Gemini Nano)

### Enable Chrome Built-in AI

1. Open `chrome://flags/#optimization-guide-on-device-model`
2. Select **Enabled BypassPerfRequirement**
3. Open `chrome://flags/#prompt-api-for-gemini-nano`
4. Select **Enabled**
5. Restart Chrome
6. Wait for model download (check `chrome://components/`)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/docuflow-ai.git
   cd docuflow-ai
   ```

2. Load in Chrome:
   - Open `chrome://extensions/`
   - Enable **Developer mode**
   - Click **Load unpacked**
   - Select the extension directory

3. Pin the extension to your toolbar for easy access
